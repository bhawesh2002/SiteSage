# SiteSage â€” Automated SEO Performance Analyzer

SiteSage is a production-grade full-stack web platform that analyzes website URLs for SEO and performance quality.  
It crawls a given website, extracts key on-page SEO signals, evaluates performance metrics, and generates structured SEO reports enriched with AI-powered insights and optimization recommendations.

This project was built as part of a Full Stack Developer technical assignment for **Madeline & Co.**, with a strong focus on clean architecture, reliability, and pragmatic AI integration.

## âœ¨ Key Features

- URL-based SEO audit with idempotent behavior
- On-page SEO analysis (titles, meta tags, headings, images)
- Lightweight performance metrics (TTFB, response time, page size)
- AI-generated SEO quality summaries and actionable recommendations
- Modern responsive dashboard for viewing audit reports
- Fully containerized backend and frontend
- CI/CD pipeline with automated builds and deployments

## ğŸ§  Design Decisions

- **Idempotent audits**: Submitting the same URL returns the existing audit instead of re-running analysis.
- **URL normalization** ensures consistent uniqueness across submissions.
- **AI output is treated as untrusted input** and defensively rendered in the UI.
- **Lightweight performance metrics** are used to avoid heavy external dependencies.
